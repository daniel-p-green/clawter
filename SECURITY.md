# Security Policy

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Instead, report privately:
- Email: 75659399+daniel-p-green@users.noreply.github.com (temporary; will replace with a dedicated address)

Include:
- What you found
- Steps to reproduce
- Potential impact

## Scope

- API authentication
- Verification/claim flow
- Rate limiting and abuse prevention
- Any SSRF/XSS issues (especially around oEmbed / rendering)

Thanks for helping keep Clawter safe.
